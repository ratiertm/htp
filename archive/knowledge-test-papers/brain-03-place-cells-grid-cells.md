# Place Cells and Grid Cells in Spatial Navigation

The hippocampal formation contains specialized neurons that construct cognitive maps
of physical environments. Place cells in CA1 and CA3 fire when an animal occupies
specific locations, with population activity uniquely identifying the current
spatial position. Grid cells in medial entorhinal cortex exhibit hexagonal firing
fields tessellating the environment, providing a metric coordinate system upstream
of place cells.

These spatial representations support self localization, path integration, and
goal directed navigation. Border cells encode environmental boundaries, head
direction cells signal facing orientation, and speed cells modulate firing with
movement velocity. Together this neural circuit implements a comprehensive cognitive
map analogous to a GPS system in the brain. Recent evidence extends place and grid
cell function beyond physical space to abstract conceptual spaces, suggesting these
cells provide a general mechanism for organizing knowledge along arbitrary
dimensions through entorhinal hippocampal interactions.
