# Complementary Memory Functions: Pattern Separation and Completion

The hippocampal memory system implements two complementary computational operations
that enable both storage and retrieval of episodic memories without catastrophic
interference. Pattern separation in the dentate gyrus orthogonalizes similar input
representations into distinct sparse codes, preventing memory confusion among related
experiences. Pattern completion in CA3 retrieves complete memory traces from partial
or degraded cues through recurrent attractor dynamics.

These two operations require fundamentally different connectivity statistics. Dentate
gyrus uses sparse mossy fiber projections to CA3 with low overlap, while CA3
recurrent collaterals form a dense associative memory network. The balance between
separation and completion is regulated by neuromodulatory state and recent
experience. Population coding through place cells encodes spatial location, with
hippocampal sequences supporting episodic memory consolidation during sharp wave
ripples in offline replay periods.
