# Working Memory and Prefrontal Cortex

Working memory refers to the limited capacity system that maintains and manipulates
information over short timescales to support ongoing cognitive operations. Classic
formulations propose a capacity of about seven items with separate stores for
verbal and visuospatial content coordinated by a central executive. Working memory
capacity correlates with fluid intelligence and predicts academic and professional
performance.

Prefrontal cortex maintains working memory representations through persistent neural
activity that bridges temporal gaps between stimulus and response. Persistent
delay period firing in dorsolateral prefrontal cortex encodes specific item
identities and locations. Recurrent connectivity within prefrontal circuits
implements attractor states that resist distraction while remaining flexible to
updating. Recent theoretical work proposes alternative dynamic coding schemes where
working memory is supported by short term synaptic plasticity rather than
persistent firing alone. Working memory provides the temporal scaffold for
sequential reasoning, language comprehension, and goal directed behavior.
