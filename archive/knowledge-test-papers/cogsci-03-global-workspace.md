# Global Workspace Theory of Consciousness

Global workspace theory proposes that conscious access arises from the broadcast of
information from local specialized processors to a global workspace accessible to
the entire brain. Unconscious processing occurs in parallel across modular neural
systems, while conscious awareness requires integration into a unified workspace
that disseminates information broadly across cortex.

The neural implementation involves long range cortical neurons forming a workspace
network spanning prefrontal, parietal, and temporal regions. Ignition refers to the
sudden global broadcast that distinguishes conscious from unconscious processing.
Working memory, voluntary attention, and verbal report depend on this workspace
mechanism. Recent neural recordings demonstrate late wave dynamics consistent with
ignition during conscious perception tasks. Global workspace theory provides a
functional framework linking specific neural events to subjective experience,
proposing that consciousness functions as a flexible all to all communication
system enabling coordinated cognitive processing across otherwise modular
subsystems of the brain.
